Нажмите кнопку Сброс, затем приступайте к решению.

  1. Выберите правильную схему расположения текста в памяти редактора (красный квадратик — знак конца строки).



  2. Выберите правильную схему расположения текста в памяти редактора (красный квадратик — знак конца строки).



  3. Разрежьте текст так, чтобы в каждой строчке находилось одно слово (со знаками препинания) без начальных пробелов. Для быстрого движения по словам используйте аккорд Ctrl+Вправо.





  4. Склейте слова так, чтобы получилось предложение, записанное в одну длинную строку.



  5. Удалите из текста все фрагменты, заключенные в угловые скобки <...> вместе с самими скобками.



  6. Наберите текст точно так, как он изображен ниже.






Чтобы увидеть результат работы, нажмите кнопку Оценка.
